Welcome to MyExtension documentation!
=====================================

MyExtension is a super awesome extension to `PyMoDAQ`__. It add many super cool features to this already super nice package :-)

__ http://pymodaq.cnrs.fr


Information
***********

GitHub repo: https://github.com/PyMoDAQ/extension_template

Documentation: http://pymodaq.cnrs.fr/

Based on PyMoDAQ and the ``pyqtgraph`` library.
